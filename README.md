## About This Project

This portfolio is my digital presence as a frontend developer and BCA student at St. Xavier's College, Ahmedabad. It was designed with one goal: to show real work, in honest context, with enough depth that the right people understand what I actually do.

## Design Decisions

**Typography-first layout** — The portfolio leads with writing before visuals because the work has real context worth communicating.

**No flashy animations** — Motion is used only where it adds clarity, not decoration.

**Pricing is public** — Transparency builds trust. Clients who see pricing upfront are better-fit clients.

**Case study depth** — Each project explains the problem, not just the output.
